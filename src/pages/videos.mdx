## About These Videos

These videos are automatically extracted from the README files of our open source repositories. They provide:

- **Hands-on Demonstrations**: See the technologies in action
- **Implementation Guidance**: Learn how to set up and use the projects
- **Real-world Use Cases**: Understand practical applications
- **Best Practices**: Discover recommended approaches

## Want to Contribute?

If you have created video content for any DecisionsDev project:

1. Add the video link to your repository's README file
2. Run `npm run build:videos` to refresh the video list
3. The video will automatically appear on this page
4. Share your knowledge with the community!

### Supported Video Platforms

- **YouTube**: Direct links or embedded videos
- **Vimeo**: Direct links or embedded videos
- **GitHub**: User-attachments videos (uploaded directly to GitHub)
