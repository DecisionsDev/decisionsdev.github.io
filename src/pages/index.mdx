## Welcome to DecisionsDev

The DecisionsDev organization hosts open source projects, tools, and samples for **IBM Operational Decision Manager (ODM)**, **IBM Automation Decision Services (ADS)**, and related IBM decision automation products. Our mission is to provide the community with high-quality resources for working with IBM's decision automation platforms.

### Getting Started

1. **Browse Repositories**: Visit our [repository browser](/repositories) to find projects by product, component, or type
2. **Clone and Explore**: Each repository includes detailed README files and documentation
3. **Contribute**: We welcome contributions! See our [contributing guidelines](https://github.com/DecisionsDev/.github)

### Repository Organization

We use a structured topic system to organize our repositories:

- **product-xxx**: Product identifier (e.g., `product-odm`, `product-decision-intelligence`)
- **comp-xxx**: Component identifier (e.g., `comp-decisioncenter`, `comp-ruleexecutionserver`)
- **type-xxx**: Repository type (e.g., `type-sample`, `type-tool`, `type-tutorial`)

Learn more in our [Topics Guide](https://github.com/DecisionsDev/decisionsdev.github.io/blob/main/TOPICS_GUIDE.md).
