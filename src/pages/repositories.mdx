### How to Use This Page

**Filter by Topics**: Use the filters above to find repositories by:
- **Product**: `odm`, `decision-intelligence`, `bai`, `cp4ba`
- **Technology**: `docker`, `kubernetes`, `java`, `business-rules`, `rpa`
- **Type**: `tutorial`, `sample`, `integration`, `tools`

**Search**: Use the search box to find repositories by name or description

**Browse**: Scroll through all repositories or click on a product card to see related projects

### Understanding Topics

Each repository is tagged with 3-7 topics that describe:
- **What product** it relates to (e.g., `odm`, `decision-intelligence`)
- **What technologies** it uses (e.g., `docker`, `java`, `kafka`)
- **What type** of resource it is (e.g., `tutorial`, `sample`, `integration`)

Common topics you'll see:
- `odm` - IBM Operational Decision Manager (most popular)
- `docker` - Container-based deployments
- `tutorial` - Step-by-step learning resources
- `business-rules` - Business rules engines
- `rpa` - Robotic Process Automation integrations

### For Contributors

If you maintain a repository in DecisionsDev, please add relevant topics:

1. **Use existing topics** - Check what topics are already in use
2. **Choose 3-7 topics** - Select the most relevant ones
3. **Add via GitHub UI** - Click the gear icon ⚙️ next to "About" on your repository page

See our [Topics Guide](https://github.com/DecisionsDev/decisionsdev.github.io/blob/main/docs/TOPICS_GUIDE.md) for detailed guidance.
